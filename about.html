About Mac App Installer

A simple way to set up your Mac with the apps you need

Created by Vali Neagu

A developer who wanted to simplify the Mac setup process

The Idea

Started as a personal tool to streamline fresh macOS installations

Now Public

Shared with the community to help others save time

Mission

To make setting up a new Mac as simple as checking a few boxes

Security

All apps are downloaded directly from their official sources

Open Source

The entire project is open source and available on GitHub
